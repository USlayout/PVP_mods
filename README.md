# PVP_mods
PVPで必要なデータパックやmodの作成用

## 作成予定mod
1. 初期ランダムスポーンmod(first random spawn)
2. 睡眠禁止 Gamerule データパック(Sleep Disable Gamerule)
3. システム時間と同期させるmod(system time mineraft)
4. プレイヤー特定mod(search server players)

### 簡易説明
1. 初期ランダムスポーンmodは全員が同じ初期リスだと困るため初期リスをプレイヤーごとにランダムにするmodです。
2. 睡眠禁止 Gamerule データパックは夜をスキップするのを禁止しまたファントムを湧かないようにするものです。ファントムの落とす素材はKubeJSかCraftTweakerで補完する。
3. システム時間と同期させるmodはマイクラの一日をシステム時間と同じにするmodです。
4. プレイヤー特定modは使用すると現在サーバーにいる全プレイヤーの位置（ディメンション＋座標）を開示するアイテムを追加するmodです。
